## About
- Twitter api clone with Java
- Access the Node.js version <a href="https://github.com/kethlynsara/projeto11-tweteroo">here</a>


## Features
- Sign up 
  - **POST** `/signup`
- Tweets
  - **GET** `/tweets`
  - **GET** `/tweets/USERNAME`
  - **POST** `/tweets`
